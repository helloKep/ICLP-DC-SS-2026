\begin{itemize}
\item We run the experiments on the environments used in the final round of the Flatland 3 competition
\item There are fifteen classes of increasing difficulty, each with ten environments
\item Three parameters are modified to adjust the level of difficulty: the number of agents, the size of the environment, and the number of cities
%\item The first and easiest class \texttt{Test\_00} contains small benchmark environments with only 7 agents
%\item The final and most difficult class \texttt{Test\_14} contains large benchmark environments with 425 agents
\item For our experiments, we use the 80 benchmark environments
\item Across these environments, the number of agents increases from 7 to 80 and the number of cities from 2 to 13
